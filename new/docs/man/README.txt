# man(1) pages for Viua VM

Use the following command to render and view them without installation:

    $ groff -man -Tutf8 ./docs/man/viua-ops.7 | less -R

If installed, the man(1) pages for all pieces of the Viua toolchain (ie, the
assembler, disassembler, repl, etc.) are displayed using the --help option.
